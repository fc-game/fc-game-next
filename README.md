
# Family Computer Game(next.js)

## Description

This is a game that I made for my family. It is a simple game that I made using next.js and react. It is a game that you can play with your family and have fun together.

## Features

- Simple game that you can play with your family
- Easy to use
- Fun to play

## How to play

1. Clone the repository
2. Install the dependencies
3. Run the game
